## Valores en el API del Show Transaction

A continuación se muestran dos campos nuevos con sus respectivos valores:

### Message Category

| Constante                | Valor   | Descripción                                         |
|--------------------------|---------|-----------------------------------------------------|
| **PA**                   | “01”    | Autorización para PAGO                             |
| **NPA**                  | “02”    | Autorización para NO PAGO                          |

### 3DS Requestor Authentication Method

| Constante                             | Valor   | Descripción                                             |
|---------------------------------------|---------|---------------------------------------------------------|
| **FRICTIONLESS_AUTHENTICATION**      | “01”    | Frictionless authentication occurred by ACS           |
| **CARDHOLDER_CHALLENGE_OCCURRED**     | “02”    | Cardholder challenge occurred by ACS                  |
| **AVS_VERIFIED**                      | “03”    | AVS verified                                           |
| **OTHER_ISSUER_METHODS**             | “04”    | Other issuer methods                                  |
