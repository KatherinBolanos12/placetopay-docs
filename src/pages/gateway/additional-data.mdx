# Datos adicionales

Un dato adicional es información extra que se agrega como parte de la transacción. Estos datos varian según factores como el proveedor de servicio de pago, franquicia o requisitos especificos de procesamiento.

## Datos en respuestas a process transaction

Existen datos adicionales que son recibidos en respuesta a la solicitud de procesamiento de una transacción. La recepción de estos datos depende de diversos factores, y viene en la respuesta como `additional`.

```json
{
    "status": {
        "status": "APPROVED",
        "reason": "00",
        "message": "Approved",
        "date": "2024-07-17T15:28:35-05:00"
    },
    // ...
    "additional": {
      // Datos adicionales
      "bin": "411076",
      "terminalNumber": "00990101",
      // ...
    }
}
```
