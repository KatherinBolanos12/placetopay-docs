### Forwarding

Forwarding es una configuración a nivel de sitio necesaria cuando un sitio cuenta con más de un medio de pago de la misma marca (Visa y Mastercard), pero con diferentes proveedores y se requiere definir un adquirente principal.

Esta funcionalidad le permite al comercio determinar qué proveedor y adquirente se debe utilizar en cada transacción con tipo de crédito corriente, asegurando que no ocurran conflictos entre los diferentes proveedores disponibles.

**Esto sólo aplica para transacciones con tarjetas Visa, Master y tipo de crédito pago corriente.**

Para solicitar esta funcionalidad comunicarse  al correo electrónico: servicioposventa@placetopay.ec

### ¿Cómo verificar si un medio de pago tiene configurado forwarding?

Al consumir el servicio **information**, la respuesta incluirá el atributo **hasForwarding**. Si este atributo tiene un valor de `true`, significa que el forwarding está configurado; en caso contrario, su valor será `false`.

Para que el valor sea `true`, deben cumplirse las siguientes condiciones:

- Contar con al menos dos proveedores de la misma marca (por ejemplo: Visa Interdin y Visa Medianet).
- Tener configuradas las reglas de selección de acuerdo a los créditos a enrutar por el adquirente.
- Tipo de crédito Corriente
- Medios de pago Visa y Mastercard

### Flujo de forwarding con API 

![Flujo](/ForwardingFlow_es.webp)

Cuando la bandera **hasForwarding** se establece en `true` en la petición **solicitud de Información**, es necesario incluir el atributo **credit** en el objeto **instrument** de las siguientes solicitudes:
- **Solicitud de Información**
- **3DS Lookup**
- **Generación de OTP**
```json
    // ...
    "instrument": {
        "card": {
            "number": "4111111111111111",
            "expiration": "12/31",
            "cvv": "303"
        },
        "credit": {
            "code": "1",
            "type": "00",
            "groupCode": "C",
            "installments": 1
        }
    }
    // ...
```

Para más información sobre **credit**, revisar la documentación de [Credit](/gateway/additional-data#additional-data-credit).
