## Búsqueda de facturas {{ tag: 'POST', label: '/invoice/search/' }} 

**Parámetros**

| Nombre    | Tipo                     |  Descripción                                                  |
|-----------| ------------------------ |-------------------------------------------------------- |
| `auth`        | Pago Corriente           | Pago a una cuota                                        |
| `agreement`   | Diferido sin intereses   | Pago por cuotas sin interés.                            |
| `searchType`  | Diferido plus            | Mezcla con o sin interes, definido por el banco emisor  |
| `searchValue `| Diferido con intereses.  | Pago por cuotas con interés.                            |
| `siteId`      | Plan pagos normal        |                                                         |
| `filters (Opcional)`  | Plan pagos especial      | Pago Sin cobro de intereses                             |
| `locale `     | Diferido preferente      | Pago por cuotas con interés más bajo                    |
