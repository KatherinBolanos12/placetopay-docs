# Micrositio Personalizado
<br/>
Este servicio web y su arquitectura permitirá a PlacetoPay hacer una consulta a un
servicio externo mediante el contrato mencionado en este documento para obtener
facturas de pago y en caso de que se realice un pago en la plataforma hacer un posterior
asentamiento del mismo.

Para usar el servicio web, deberás tener un lenguaje de programación que pueda
comunicarse con un servicio REST.

El método de integración contempla las operaciones básicas para consulta de facturas
disponibles por referencia, documento o referencia alterna y el posterior asentamiento del
pago realizado

A continuación, se describirán cada uno de los usos y los datos requeridos en la trama
para cada uno de los casos. La respuesta para cada operación está unificada en un solo
tipo de trama resultante.


### ¿Cómo funciona? 

Se realizan consumos individuales a cada uno de los endpoints de acuerdo con la
necesidad.
