Una vez dentro del micrositio, la URL del navegador, es la que se debe compartir con el cliente **(siempre es la misma para todos los clientes)** y de esta manera puedan ingresar los datos en los campos del micrositio estandar:

-	**Referencia:** este campo se entiende como la referencia de la transacción y es recomendado que sea única por proceso de pago.
-	**Descripción del pago:** Un detalle breve de la compra.
-	**Monto:** valor del producto o servicio.
-	**Documento del comprador:** Es el número de documento de cliente final.
-	**Nombre del comprador:** Es el nombre del cliente final.
-	**Apellido del comprador:** Es el apellido del cliente final.
-	**Correo electrónico del comprador:** Es el correo del cliente final

<Note> Estos campos los deben llenar los clientes finales </Note>

La URL selecciona es la cual se debe compartir con el cliente
