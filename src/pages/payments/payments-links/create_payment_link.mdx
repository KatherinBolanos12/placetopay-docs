# Creación de un Link de Pago

Para crear un nuevo link de pago se debe dar clic en el icono **+**  {{ className: 'lead' }}

Posterior se visualizarán los siguientes campos que son generalidades de un link de pago, los cuales deben ser diligenciados de la siguiente manera:

**Vincular a:** En este item se selecciona si el link de pagos va a estar atado a un sitio o micrositio, en este campo siempre elegir la opción Sitio.

<Note> En caso de tener alguna configuración especial, se deberá asociar a un micrositio. </Note>

**Sitio:** Selecciona el nombre del sitio al que se va a vincular el link de pago que se está creando, para este caso se despliega solo los sitios asociados al comercio.

**Nombre:** Escribir el nombre con el que va a aparecer el link de pago en el histórico de link de pagos creados. Así se llamaría el link de pagos que se va a compartir.

**Número de pagos permitidos:** En esta opción se puede ingresar la cantidad de pagos aprobados que se va a permitir con dicho link de pago. **(Al dejarlo vacío, los pagos aprobados serán ilimitados hasta que llegue a la fecha de expiración)**.

**Tiempo de expiración del proceso de pago:** Es el tiempo **(en minutos)** para que el proceso de pago expire, este tiempo inicia desde que en el link de pago da clic en pagar y comienza el flujo transaccional de solicitud de datos al pagador en el Checkout y se recomienda que sea de 10 a 15 minutos.  

**Fecha de expiración:** Es la fecha límite que se tiene para utilizar el link de pago, es decir, el tiempo de vigencia, después de esta fecha, el link de pago se inhabilitará.

Luego de diligenciar todos los campos se debe presionar el botón siguiente. {{ className: 'lead' }}

**Referencia:** Esta referencia es la identificación de la transacción y es el valor con el cual se identificará la transacción realizada en la consola administrativa de Evertec Placetopay. Este campo puede ser un valor alfanumérico que se le asignará al link de pago.

<Note> Este campo no permite caracteres especiales (solo guion medio) y permite un máximo de 32 caracteres </Note>

**Descripción:** Un detalle breve de la compra. Valor Alfanumérico de largo max 250

**Moneda:** Se ingresa el tipo de moneda con el que se va a generar el link de pago y se habilitará en el flujo de pago **(En caso de que tengan habilitado múltiples monedas en el panel administrativo)**

**Monto:** Se ingresa el valor total que se va a pagar con este link de pagos.

**Correos electrónicos de los destinatarios:** Se ingresa el o los correo a los cuales se les va a enviar el link para el pago **(este campo no es obligatorio)**. En caso de requerir enviar a varios correos electrónicos, al momento de completar el primer correo, podrá dar enter y se habilitará el ingreso de correos adicionales.
