# Proceso de pago

Aquí encontrarás todo lo necesario para gestionar tus pagos de forma fácil y eficiente. Te ofrecemos una guía paso a paso para completar tus transacciones, consultar su estado y acceder al historial de manera clara y sencilla.{{ className: 'lead' }}

## Email {{ id: 'functions', tag: '', label: '' }}

Una vez compartido el enlace a través del correo, el cliente podrá visualizarlo de la siguiente forma, tal como se ilustra a continuación: {{ className: 'lead' }}


<div className="w-full sm:w-4/4 lg:w-2/3 sm:ml-10 sm:mr-6 flex justify-center">

Mediante el contenido que llega al correo, se permite realizar transacciones de forma rápida. Se cuenta con un código QR para facilitar el proceso de pago y se puede acceder al pago mediante el enlace ubicado en la parte inferior. Además, se incluye un resumen de pago como el monto, la referencia y la fecha límite.{{ className: 'lead' }}
<Note>
    La URL incluida en el correo es de uso único para el correo enviado. Si se requiere compartir el enlace de pago con varias personas, recuerde compartirlo mediante las opciones mencionadas anteriormente.
</Note>

## Vista de pago {{ id: 'functions', tag: '', label: '' }}

Al abrir el enlace de pago, ya sea a través del correo electrónico, el código QR, una URL directa o un mensaje de WhatsApp, se mostrará la siguiente información: la fecha de expiración, el valor a pagar, el comercio al que se realizará el pago, la descripción del pago, la referencia y el estado del enlace de pago.{{ className: 'lead' }}

## Proceso Webcheckout{{ id: 'functions', tag: '', label: '' }}
Posteriormente, el cliente es redirigido al proceso de pago (checkout), donde podrá visualizar nuevamente el resumen del pago a realizar, ingresar su correo electrónico y hacer clic en el botón 'Continuar'.{{ className: 'lead' }}
<div className="flex justify-center">

Después de ingresar el correo electrónico y hacer clic en 'Continuar', se procederá a ingresar sus datos de pago y hacer clic en el botón 'Pagar'. Es importante tener en cuenta que, estando en este apartado, existe un tiempo de expiración, el cual se indica en la esquina inferior derecha.{{ className: 'lead' }}
