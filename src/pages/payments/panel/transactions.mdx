## Módulo de transacciones

Al ingresar al módulo de transacciones, encontrarás un filtro de búsqueda rápida que te permitirá localizar una transacción utilizando:
- La referencia de la transacción.
- El correo electrónico del usuario que realizó la compra.

<CenteredContainer>
     ![search transactions](/payments/panelold/imagen1.webp)  
</CenteredContainer>

Si no dispones de estos datos, puedes acceder al apartado **FILTRO**, donde encontrarás diversos métodos de búsqueda para localizar la transacción como los siguientes:

En el apartado **Operador**, encontrarás tres métodos de búsqueda: **Igual a**, **Diferente de** y **Como**. 
Es importante recordar que, después de seleccionar uno de estos valores, colocar en el campo valor lo correspondiente, 
debes hacer clic en **Agregar condición** para que la selección se refleje en el campo de filtros seleccionados.
Puedes aplicar tantos filtros como consideres necesarios para optimizar tu búsqueda

<CenteredContainer>
     ![filters](/payments/panelold/imagen4.webp)  
</CenteredContainer>

También puedes realizar las siguientes acciones:

- **Remover condición**: Elimina un solo filtro aplicado.
- **Limpiar todas**: Borra todos los filtros seleccionados.
- **Guardar filtro**: Guarda la configuración actual de filtros para futuras búsquedas..

## Estadisticas de transacciones

En este módulo, podrás analizar las estadísticas y el comportamiento transaccional de tu sitio o comercio, 
lo que te permitirá obtener información clave sobre las operaciones realizadas y tomar decisiones informadas para optimizar tu rendimiento.

<CenteredContainer>
     ![filters](/payments/panelold/imagen6.webp)  
</CenteredContainer>

Esta herramienta ofrece diversos tipos de informes, según lo que desee el cliente.
Como pasarela, recomendamos el **Resumen General**, ya que es el más completo y proporciona una visión integral de las transacciones.

Podrás seleccionar distintos tipos de datos para visualizar el comportamiento de las transacciones:
- **Fecha inicial y fecha final**: Rango de tiempo en el que deseas analizar las transacciones.
- **Período**: Puedes filtrar por día, mes, trimestre, año, hora del día o día de la semana.
- **Aerolínea**: Aplica solo para transacciones relacionadas con aerolíneas; en otros casos, deja este campo vacío.
- **Respuesta**: Filtra por un tipo específico de respuesta (aprobada, rechazada, declinada, etc.). Si deseas ver todas las transacciones, deja este campo vacío.
- **Comercio**: Especifica el nombre del comercio al que pertenece la transacción.
- **Sitio**: Selecciona el sitio del cual deseas generar el reporte.
- **Moneda**: Útil si manejas transacciones en múltiples divisas.
Una vez definidos estos valores, simplemente haz clic en el botón **Generar** para obtener el reporte.
