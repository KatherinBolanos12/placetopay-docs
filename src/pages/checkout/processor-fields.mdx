# Datos de procesamiento

La propiedad `processorFields` son datos adicionales que son agregados en el procesamiento de la transacción, y estos datos se encuentran en `payment.processorFields`.

```json
{
    "requestId": 7,
    "status": { /* ... */ },
    "request": { /* ... */ },
    "payment": [
        {
            "processorFields": [
                {
                    "value": "00990101",
                    "keyword": "terminalNumber",
                    "displayOn": "none"
                },
                {
                    "value": "529259",
                    "keyword": "bin",
                    "displayOn": "none"
                },
                {
                    "value": "0745",
                    "keyword": "lastDigits",
                    "displayOn": "none"
                },
                {
                    "value": "b466b12c2a248122c9448783c77435ed",
                    "keyword": "id",
                    "displayOn": "none"
                },
                {
                    "value": "000",
                    "keyword": "b24",
                    "displayOn": "none"
                },
                /// ...
            ],
            // ...
        }
    ],
}
```

Adicional a estos datos, existen otros como:

Clave                   | Ejemplo                               | Descripción                     
------------------------|---------------------------------------|------------------------
`id`                    | "b466b12c2a248122c9448783c77435ed"    | Identificador de la sessión
`b24`                   | "000"                                 | Estado de la transacción en ISO 24
`credit`                | ```{"type": "00", "groupCode": "C", "code": "0", "installments": 0}```    | Datos de credito.
`totalAmount`           | 63.58                                 | Monto total.
`interestAmount`        | 5.78                                  | Monto de interés.
`installmentAmount`     | 2.65                                  | Monto de la cuota.
`iceAmount`             | 0                                     | Monto de ICE. 
`line`                  | null                                  | Línea.
`cardType`              | "credit"                              | Tipo de tarjeta.
`trazabilyCode`         | 999900                                | Código de trazabilidad.
`transactionCycle`      | 3                                     | Ciclo de transacción.
`RequestId`             | "1560778247"                          | ID de solicitud.
`PartnerAuthCode`       | "5d029607e1abb"                       | Código de autorización de socio.
`merchantCode`          | "4682312740123"                       | Código de negocio.
`terminalNumber`        | "00990101"                            | Número de terminal.
`bin`                   | "411076"                              | Bin, primeros digitos de la tarjeta.
`bankName`              | "Bank Name"                           | Nombre de banco.
`accountNumber`         | "1111"                                | Número de cuenta.
`expiration`            | "1222"                                | Expiración de la tarjeta en formato (mes)(año).
`installments`          | "2"                                   | Número de cuotas.
`pocket`                | "Subsidio Educativo"                  | Bolsillo.
`fiscalControlNumber`   | "Control: MYHFC-KEHKC"                | Número de control fiscal.
`fiscalControlProvider` | "ET"                                  | Provedor de control fiscal, de ejemplo ET como Evertec.
