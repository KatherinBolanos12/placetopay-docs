# Campos

La propiedad Campos `fields` es una estructura utilizada para almacenar y mostrar en la Interfaz información relevante relacionada a productos o artículos en el proceso de cobro. Puede contener varios objetos [NameValuePair]('/checkout/nameValuePair') que representan diferentes datos a anexar.
Más detalles en [NameValuePair]('/checkout/nameValuePair')

## Ejemplo

```json
{
...
"fields": [
    {
      "keyword": "cmsInvoiceId",
      "value": "string",
      "displayOn": "both"
    }
  ]
}
```

## Resultados

- Cuando envías múltiples [NameValuePair]('/checkout/nameValuePair') con el mismo `keyword`, nuestra API elige el primero para almacenar y mostrar en la interfaz.
- La propiedad `displayOn` te brinda control sobre cuándo un [NameValuePair]('/checkout/nameValuePair') se mostrará o no en diferentes vistas del proceso. 
Puedes usar esto para decidir si deseas que ciertos datos se vean durante la captura de datos, en el resultado del proceso, en ambas etapas o nunca. Más detalles en [NameValuePair]('/checkout/nameValuePair')
