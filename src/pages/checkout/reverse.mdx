# Reembolsos
Puedes realizar reembolsos de las transacciones, previamente aprobadas, utilizando la referencia interna `payment.internalReference`, la cual se encuentra en la respuesta al [consultar una sessión](/checkout/api/reference/session#query-a-session).

## ¿Cómo realizar un reembolso?
Para realizar un reembolso se debe hacer una petición al endpoint [Reembolso](/checkout/api/reference/payment#reverse-payment). Existen dos tipos de reembolsos *total* o *parcial*:

- **Reembolso total** : 
Se revierte el monto total de la transacción aprobada.

- **Reembolso parcial** : 
Se revierte solo una parte del monto total de la transacción aprobada. Para realizar reembolsos parciales, se debe enviar la propiedad `amount` en la petición. Estos reembolsos se pueden realizar tantas veces como sea necesario hasta que se haya completado el monto total de la transacción.  
Una vez que una transacción se ha reembolsado por completo, no es posible realizar otro reembolso. Si se realiza una petición de reembolso sobre una transacción completamente reembolsada, este método devolverá un error  
`status.status.message` `La transacción ya ha sido reembolsada`

<Note>
    Un reembolso parcial solo es procesado si la transacción ya ha sido conciliada en red bancaria.
</Note>

## Verificar una transacción reembolsada
Para verificar que una transacción fue reembolsada total o parcialmente, se debe [consultar la sesión](/checkout/api-reference/session#query-a-session) de la transacción original donde encontraras en la respuesta la propiedad `payment.refunded` con valor `true`.
