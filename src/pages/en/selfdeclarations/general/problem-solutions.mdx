# Solución a posibles problemas
A continuación se muestran algunos casos que se podrían presentar en el registro de usuarios, principalmente aquellas situaciones en las que se envía un correo electrónico al usuario y no puede ingresar al sistema.

1. Cuando el usuario presiona Completar Registro y no lo lleva al sistema, se recomienda copiar la URL que se muestra en la imagen, pegarla en el navegador para que sea redireccionado al sistema.

2. Si el usuario al seleccionar el botón Ir a mi perfil tiene problemas de conexión, podrá copiar la URL que se muestra en la imagen en su navegador para acceder.

3. En caso que un usuario olvide su contraseña la podrá restablecer dando clic en ¿Olvidaste tu contraseña? en la pantalla de inicio de sesión.


Se ingresa el correo electrónico registrado por el usuario, se da clic en Restablecer y el sistema enviará un correo para restablecer la contraseña.

4. Un usuario declarante en caso de que pierda la llave electrónica puede reexpedir una nueva. En la pantalla de ingreso debes seleccionar el link que dice Perdiste tu llave electrónica?

Posteriormente el usuario debe ingresar un correo y luego seleccionar el botón Solicitar reexpedición para continuar con el proceso de reexpedir llave electrónica.

Inmediatamente se envía un mensaje al correo ingresado para confirmar el proceso de reexpedir llave electrónica donde el usuario puede continuar seleccionando el botón Reexpedir llave electrónica o ignorar el mensaje.

Luego el usuario debe confirmar la reexpedición completando los datos que se muestran en el formulario donde se creará una petición de reexpedición de llave electrónica la cual debe ser aprobada por un administrador del servicio.

Finalmente se genera un mensaje informando sobre la solicitud, la cual una vez aprobada recibirás un correo electrónico para generar nuevamente tu llave electrónica.

5. Cuando se genera un mensaje que no permite completar el registro de un declarante:

> ¡Upss!, algo está mal, por favor revisa tus datos

Cualquiera de los datos del formulario es incorrecto. Debe tenerse en cuenta que el usuario autorizado debe estar relacionado con la empresa para la declaración específica que se escoge en el formulario. Si existe el usuario pero no tiene el permiso a la renta específica no podrá realizar el proceso.

6. Cuando ya se ha revocado una llave electrónica y al declarante no le llega la notificación por correo para descargarlo nuevamente y configurarlo y el administrador tiene la opción Reexpedir llave electrónica para usuario deshabilitada en el módulo de Usuarios Autorizados:

Para esto se debe editar el usuario autorizado que se quiere reenviar la notificación por un administrador del sistema y este debe seleccionar el botón Reenviar notificación o Reenviar enlace sin confronta, según sea el caso.

7. En caso de que se tengan varios certificados configurados en el equipo de cómputo y en el momento que se estaba ingresando se seleccionó el certificado que no corresponde al usuario, se debe ingresar a la configuración del navegador a borrar el historial, cookies, caché y se debe reiniciar el navegador para poder acceder, de lo contrario el sistema generará un mensaje de error.

Se recomienda para evitar conflictos en caso de que configure más de un certificado en un equipo de computo utilizar [Firefox](https://www.mozilla.org/es-ES/firefox/new/).

8. Cuando debes realizar un pago y no se ha autorizado la declaración para realizar la firma electrónica porque no es visible, se debe comunicar con el administrador del sistema para que habilite la opción.
