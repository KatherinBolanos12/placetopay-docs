# Processor fields

The `processorFields` property are additional data fields added during the transaction processing and these data are found in `payment.processorFields`.

```json
{
    "requestId": 7,
    "status": { /* ... */ },
    "request": { /* ... */ },
    "payment": [
        {
            "processorFields": [
                {
                    "value": "00990101",
                    "keyword": "terminalNumber",
                    "displayOn": "none"
                },
                {
                    "value": "529259",
                    "keyword": "bin",
                    "displayOn": "none"
                },
                {
                    "value": "0745",
                    "keyword": "lastDigits",
                    "displayOn": "none"
                },
                {
                    "value": "b466b12c2a248122c9448783c77435ed",
                    "keyword": "id",
                    "displayOn": "none"
                },
                {
                    "value": "000",
                    "keyword": "b24",
                    "displayOn": "none"
                },
                /// ...
            ],
            // ...
        }
    ],
}
```
