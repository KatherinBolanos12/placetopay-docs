# Processor fields

The `processorFields` property provides additional information about the processing of a completed session. Each object is stored in `payment.processorFields` with the type [NameValuePair](additional-fields#name-value-pair).

```json
{
    "requestId": 7,
    "status": { /* ... */ },
    "request": { /* ... */ },
    "payment": [
        {
            "processorFields": [
                {
                    "value": "00990101",
                    "keyword": "terminalNumber",
                    "displayOn": "none"
                },
                {
                    "value": "529259",
                    "keyword": "bin",
                    "displayOn": "none"
                },
                {
                    "value": "0745",
                    "keyword": "lastDigits",
                    "displayOn": "none"
                },
                {
                    "value": "b466b12c2a248122c9448783c77435ed",
                    "keyword": "id",
                    "displayOn": "none"
                },
                {
                    "value": "000",
                    "keyword": "b24",
                    "displayOn": "none"
                },
                /// ...
            ],
            // ...
        }
    ],
}
```

## Description of the `Credit` object  {{ id: "description-credit" }}

The possible properties of the `credit` object and the values that can be found in them are described below:

### Property `type`

The `type` property defines the type of the credit and the following values can be found:

| Type                                              | Description                  |
|---------------------------------------------------|------------------------------|
| `00`                                              | Corriente                    |
| `01`                                              | Diferido Corriente           |
| `02`                                              | Diferido propio              |
| `03`                                              | Plan pagos especiales        |
| `11`                                              | Plan pagos normal            |
| `12`                                              | Diferido corriente con interés|
| `21`                                              | Diferido plus cuotas         |
| `22`                                              | Diferido plus pago total     |

### Property `groupCode`

The `groupCode` property defines the group code and is used to define the description field and the following values can be found:

| groupCode                                        | Description                  |
|---------------------------------------------------|------------------------------|
| `C`                                              | Corriente                    |
| `D`                                              | Diferido Corriente           |
| `P`                                              | Diferido propio              |
| `X`                                              | Plan pagos especial          |
| `N`                                              | Plan pagos normal            |
| `I`                                              | Diferido corriente con interés|
| `M`                                              | Diferido plus cuotas         |
| `T`                                              | Diferido plus pago total     |
| `L`                                              | Diferido preferente          |

### Property `code`

The `code` property defines the identifier of the array, and its possible values are 0 or 1.

### Property `installments`

The `installments` property defines the number of installments allowed in the credit.
