# Localización

También puedes personalizar el idioma de los links de pago, decidiendo en qué idioma se enviará y mostrará el enlace de pago al usuario {{ className: 'lead' }}

Para elegir el idioma debes enviar el parámetro `locale` en las peticiones que hagas, este tambien es el idioma inicial que tendrá la sesión.

```json
{
    "locale": "es_CO",
    ...
}
```

El parámetro `locale` es un string que contiene el código de localización, por ejemplo `es_CO`.  Es un parámetro opcional, en caso de que no se envíe o no cumpla con los estándares previamente descritos, se tomará por defecto el locale y país del sitio que está creando el link.

Ejemplos:

| Locale  |  Idioma      | País                 |
|:--------|:-------------|:---------------------|
| `es_CO` | Español (es) |  Colombia (CO)       |
| `es_EC` | Español (es) |  Ecuador  (EC)       |
| `es_PR` | Español (es) |  Puerto Rico (PR)    |
| `en_US` | Inglés  (en) |  Estados Unidos (US) |

Este parámetro se compone por el código del lenguaje según el estándar [ISO 631-1](https://en.wikipedia.org/wiki/List_of_ISO_639-1_codes), seguido de un guión bajo `_` y 2 letras que corresponden al país según el estándar [ISO 3166-1 alpha-2](https://es.wikipedia.org/wiki/ISO_3166-1#C%C3%B3digos_ISO_3166-1).

Actualmente se soportan los idiomas: Español, Ingles, Portugués e Italiano.
