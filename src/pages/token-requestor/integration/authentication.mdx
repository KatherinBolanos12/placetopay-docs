# Autenticación

Solicite a su administrador de cuenta el token para realizar la autenticacion de las APIS descritas a continuacion

### Autenticación con Bearer Token

El token para la autenticación sera proporcionado por un agente autorizado de la compañia el cual debe ser usado posteriormente en cada llamada al API a través de la cabecera HTTP “Authorization: Bearer **token** ”.
